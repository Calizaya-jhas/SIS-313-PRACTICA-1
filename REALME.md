# 📘 SIS-313 - Práctica 1

Este proyecto corresponde a la **Práctica 1 de la materia SIS-313**, donde se desarrolla una página web que demuestra el uso de tecnologías básicas de desarrollo frontend como HTML, CSS y posiblemente JavaScript.

## 🌐 Descripción

La página presenta una estructura web simple y funcional, diseñada para:

- Mostrar contenido organizado y claro
- Aplicar estilos visuales atractivos
- Servir como base para futuras prácticas y mejoras

Este proyecto es ideal para comprender los fundamentos del desarrollo web.
